大家建自己的包就建为applications.自己的代号
比如我建的就是applications.wzc
applications.wzc 和applications.test是一我写出的例子

注意事项：1.当你的应用程序涉及到多个线程共享一个资源的时候，
		就对同步问题多考虑下。
		2.大家尽量不要在action（）中加入循环很久的代码，
			这样控制器来就会很慢，而如果有死循环就无法控制了
		3.你的应用程序初始化的时候应该给从AbstractApplication
			继承过来的app成员变量进行初始化
		4.导入jar包步骤：先把jar包加入到lib文件夹中
		然后再点击项目右键Buildath->ConfigueBuildPath->
		选择Libaraies->addJars->选择jar包
